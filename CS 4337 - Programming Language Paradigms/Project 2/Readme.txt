- Explain how to compile and run your project
	1. Open terminal or Command Prompt
	2. Navigate to the Project Directory using cd command
	3. Start the Prolog Interpreter by typing "swipl"
	4. Load the Prolog Files by typing consult(fileName.pl)
		- consult('find_exit.pl').
		- consult('example.pl').
		- consult('test.pl').
	5. Run Test Cases
	6. To Exit, type "halt."
